loop & function

for loop

إختصار لتكرار أكشن معين لمجموعة عناصر ب دالة واحدة
يمكن استخدامها عدة مرات وبأوقات مختلفة دون النطر إلى الشروط وتعتبر الأكثر إستخداماً

for/in

تكرار شيء من شيء
تستخدم مع object
لا يفضل استخدامها بكثرة ويمكن استبدالها ب
(.forEach)

for/of

تكرار شي من شيء
تستخدم مع array, string, map ,NodeLists
لاتستخدم مع object

do/while

ينفذ الكود ثم يتحقق من الشرط اذا كان الشرط صحيح فإته يكرر الاكشن ولا يتوقف عن التتكرار مادام الشرط صحيح

while

ينفذ التكرار فقط في حالة كون الشرط صحيح
يعطي نفس النتيجة التي تعطيها for loop اذا كان الشرط صحيح

for loop syntax

for (statement 1; statement 2; statement 3) { // code block to be executed }

Example

for (var i = 0; i < 5; i++){ text +="The number is " + i + "
" ; }

for loop syntax

for (key in object) { // code block to be executed }

Example

var person = {fname:"John", lname:"Doe", age:25};
var txt = "";
for (let x in person) { txt += person[x] + " "; } document.getElementById("demo").innerHTML = txt;

جرب الان

for/of syntaxdo

for (variable of iterable) { // code block to be executed }

Example

var cars = ["BMW", "Volvo", "Mini"]; var text = "";
for (let x of cars) { text += x + "
"; } document.getElementById("demo").innerHTML = text;

do/while syntax

do { // code block to be executed } while (condition);

example

var text = "" var i = 0;
do { text += "
The number is " + i; i++; } while (i < 10); document.getElementById("demo").innerHTML=text;

while syntax

while (condition) { // code block to be executed }

example

var text = ""; var i = 0;
while (i < 10) { text +=" The number is " + i; i++; } document.getElementById("demo").innerHTML=text;

Declare Function

دالة تعرف كخطوة أولى ثم تستخدم لاحقاً

Expression Function

دالة تعرف من خلال متغير وتخزن داخل var
عادة تكون بدون اسم فتسمى anonymous function

Function() Constructors

دالة تخزن داخل اسم خاص بها مثل myFunction

self invoking function

دالة تستدعي نفسها من خلال حصرها بقوسين ()

Anonymes Function

دالة لا تحتوي على اسم تغرف نفسها ب function() مباشرة

Declare Function syntax

function functionName(parameters) { // code to be executed }

Example

var x = myFunction(4, 3); document.getElementById("demo").innerHTML = x; function myFunction(a, b) { return a * b; }

جرب الان

Expression Function syntax

var x = function (a, b) {return a * b};

Example

var x = function (a, b) {return a * b}; document.getElementById("demo").innerHTML = x(4, 3);

جرب الان

Function()constructor

var myFunction = new Function("a", "b", "return a * b");

Example

var myFunction = function (a, b) {return a * b} document.getElementById("demo").innerHTML = myFunction(4, 3);
